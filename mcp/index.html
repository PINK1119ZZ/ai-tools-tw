什麼是 MCP？3 句話搞懂

MCP（Model Context Protocol）是 Anthropic 在 2024 年底推出的開放標準，解決了 AI 工具「只能聊天、不能做事」的根本問題。

簡單說：以前你要讓 AI 存取 GitHub，要自己寫整合；現在只要接上 GitHub MCP Server，Cursor / Claude Code / Windsurf 全都能用，不用重複開發。

你 → AI 助理（Cursor / Claude Code）
↕ MCP Protocol（標準化協議）
MCP Server → GitHub / 資料庫 / 瀏覽器 / 搜尋 / Slack…
一次寫好，所有 AI 客戶端通用

目前（2026 Q2）已有 5,000+ MCP Server 可用，從 GitHub、PostgreSQL、Puppeteer 到 Stripe，幾乎你能想到的工具都有人做好了。

⚡ 5 分鐘快速開始（Cursor 3 示範）

以 GitHub MCP Server 為例，在 Cursor 3 裡設定 MCP

1

開啟 Cursor 設定

按 Cmd/Ctrl + , → 搜尋 「Tools & MCP」→ 點 「Add MCP Server」

2

填入 Server 設定

Name: github，Command: npx -y @anthropic-ai/mcp-server-github，Env: GITHUB_PERSONAL_ACCESS_TOKEN=你的Token

3

產生 GitHub Token

GitHub Settings → Developer Settings → Personal Access Token → 勾選 repo 權限 → 複製 Token

4

測試

在 Cursor Chat 輸入「列出我在 GitHub 上的所有 repo」，看看 AI 能不能正確回應
