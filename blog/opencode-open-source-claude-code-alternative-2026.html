OpenCode 是什麼？為什麼 153K Stars？

2026 年 AI coding agent 市場競爭激烈，但大多數工具有三個共同問題：月費 $15–50、綁定單一模型供應商、程式碼上傳到你無法控制的伺服器。OpenCode 是目前最受歡迎的開源解法。

由 Anomaly 團隊開發，OpenCode 在 GitHub 累積超過 153,000 顆星，是開源 AI coding 工具中排名第一。MIT 授權意味著你可以自由使用、修改、甚至商業部署，完全不需要付授權費。

核心定位：OpenCode 是 Claude Code 的開源替代方案。同樣在終端機運作、同樣支援 agentic 模式自主完成任務，但你可以接任何模型的 API，程式碼不離開你的機器。

OpenCode 支援三種使用介面：

  • 終端機 TUI：最輕量，適合 SSH 遠端開發
  • 桌面 App（Beta）：macOS / Windows / Linux，圖形介面
  • IDE 擴充：VS Code、Neovim、Zed 等主流編輯器

5 分鐘安裝教學

macOS / Linux（推薦方式）

# 一鍵安裝腳本
curl -fsSL https://opencode.ai/install | bash

# 或用 Homebrew
brew install anomalyco/tap/opencode

# 驗證安裝
opencode --version

Windows

# Chocolatey
choco install opencode

# 或 Scoop
scoop install opencode

# 或 npm（跨平台）
npm install -g opencode-ai

Docker（伺服器部署）

docker run -it --rm ghcr.io/anomalyco/opencode

初始化專案

# 進入你的專案目錄
cd /path/to/your-project

# 啟動 OpenCode
opencode

# 在 TUI 內執行初始化（自動分析專案結構）
/init

# 連接 API 供應商
/connect
💡 台灣開發者提示：執行 /init 後，OpenCode 會自動在專案根目錄建立 AGENTS.md，記錄專案架構與 coding 慣例，讓 AI 更準確理解你的程式碼。

核心功能深度解析

1. Multi-Session 並行執行

OpenCode 支援同時開啟多個 Agent session，可以讓不同 Agent 並行處理不同模組。例如：一個 Agent 寫後端 API、另一個同時寫前端元件，最後合併結果。

2. MCP（Model Context Protocol）整合

完整支援 MCP，可以連接 Zapier、Notion、Linear、GitHub 等外部工具，讓 Agent 直接操作你的工作流程。

3. AGENTS.md 專案記憶

執行 /init 後自動生成 AGENTS.md，記錄：

  • 專案架構與技術棧
  • Coding 風格與命名慣例
  • 常用指令與部署流程
  • 注意事項與禁止操作

4. 完全隱私保護

OpenCode 官方聲明：不儲存任何程式碼或 context 資料。搭配本地 Ollama 模型，程式碼完全不離開你的機器，適合醫療、金融、法律等隱私敏感產業。

總評與推薦對象

⭐ OpenCode 總評

✅ 優點

  • 完全免費（MIT 開源）
  • 75+ 模型自由切換
  • 程式碼不離開本機
  • 支援本地 Ollama 模型
  • 終端機 + 桌面 + IDE 三合一
  • Multi-session 並行執行
  • 完整 MCP 整合
  • Docker 自架支援

❌ 缺點

  • 桌面 App 仍在 Beta
  • 需要自行管理 API keys
  • 社群支援不如 Claude Code 官方
  • 部分進階功能文件不完整

推薦給：在意費用控制的開發者、需要隱私保護的企業、想靈活切換模型的進階用戶、有自架需求的團隊。

不推薦給：想要最簡單開箱即用體驗、不想管理 API keys、或深度依賴 Anthropic 生態系的用戶。
