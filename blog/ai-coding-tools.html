實測心得(寫一個 LINE Bot 範例)

Cursor 🥇

Composer 模式可以一次改多檔,Tab 補全準到嚇人。最舒服。缺點:吃 RAM、貴。

Claude Code 🥈

CLI 操作、Agent 自己跑 npm install / git commit。覺得自己像個經理在指揮。缺點:沒 IDE,適合熟 terminal 的人。

GitHub Copilot 🥉

穩定、便宜、整合 GitHub。缺點:沒 agent 模式,還是「補全」為主。

Windsurf

免費版很香、Cascade 模式不錯。缺點:企業功能少。
